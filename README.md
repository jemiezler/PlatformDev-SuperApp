
# Platform Development - Super App

Before Getting Start: Install necessary tool
- PNPM | https://pnpm.io/installation



## Installation

For first time, install dependencies in root directory

```bash
  pnpm install
```


## Setting Up

For first time, install dependencies in root directory

```bash
  app setup
```
Development with
```bash
  app dev
```
Or you can use for manual guides
```bash
  app help
```
